# Fashion visual design & editorial direction

**Purpose:** How fashion/apparel sites should look and feel — the visual rules that make a clothing brand read as desirable and "expensive" rather than generic.

**Harvest / review:** 2026-06

---

## Principles

| Principle | Practice |
|-----------|----------|
| **Image leads, type supports** | The hero is a photo, a painterly image, or bold type — never a paragraph. Let imagery carry the emotion. |
| **Restraint = premium** | Whitespace, few elements, one focal point per view. Dior/Burberry/Chanel are mostly empty space + one image. Clutter reads cheap. |
| **Bold minimalism** | 2026's dominant direction: clean minimal structure + high contrast + attitude. Minimal foundations, *one* loud move (oversized type, a single accent). |
| **New editorial** | Treat each screen like a magazine spread — generous margins, considered type pairing, art-gallery pacing (Rachel Comey, Anine Bing). |
| **Expressive, voiced type** | Move away from safe neutral sans. A display face with character (grotesk, condensed, or a distinct serif) sets the brand tone before a word is read. |
| **Disciplined color** | A tight palette — often B&W or near-mono + one accent. 2026 palettes lean warm off-whites/creams (not stark `#fff`), muted earth tones, deep teals. |
| **Motion with restraint** | Smooth, subtle (Burberry), not animation-heavy. Scroll reveals and slow fades, never carousels that fight the user. |

---

## Patterns

- **Hero**: full-bleed image OR oversized typographic statement; minimal nav; one CTA. Mobile-first (most fashion traffic is phone).
- **Type pairing**: one expressive display face (headlines, brand voice) + one quiet workhorse (body, UI). A mono accent reads technical/streetwear.
- **Off-white over pure white**: warm cream backgrounds (`#f4f1ea`-ish) feel considered; `#ffffff` feels like a default template.
- **Grid as gallery**: product/lookbook grids with generous gutters; let a few images break the grid for editorial rhythm.
- **Streetwear vs. luxury dialects**: streetwear = high contrast, near-black, condensed/heavy type, mono accents, a single hot accent color, drop language. Luxury = airy, serif or refined sans, muted palette, lots of air. Match the dialect to the brand.

---

## Anti-patterns

| Avoid | Why |
|-------|-----|
| Generic stock hero | Highest single conversion-killer on a launch page (waitlist research). Cheapens instantly. |
| Pure `#ffffff` + default system font | Reads as an unfinished template, not a brand. |
| Many competing focal points | Fashion needs one hero idea per view; competing elements flatten desire. |
| Rainbow palettes / many accents | Undermines the "expensive" signal. One accent, used sparingly. |
| Carousels / autoplay clutter | Fights the editorial calm; hurts a11y and focus. |
